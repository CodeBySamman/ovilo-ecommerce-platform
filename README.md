# Ovilo - Full Stack eCommerce Platform



## About Project

Ovilo is a modern full-stack eCommerce platform designed to provide a complete online shopping experience.

Users can explore products, browse collections, manage their cart, place orders, and securely authenticate using Google Login.

The platform also includes a powerful admin dashboard for managing products, orders, customers, and store operations.

---

## Features

### User Features

- User authentication with NextAuth
- Google Login integration
- Browse products by categories
- Product collections and deals
- Add products to cart
- Update cart quantity
- Remove cart items
- Order placement system
- Responsive design for mobile and desktop


### Admin Features

- Admin dashboard
- Product management
  - Add products
  - Edit products
  - Delete products
- Order management
- Customer management
- Category management
- Store overview dashboard
